# Todo App - Fullstack Architecture

A fullstack task management application implementing professional patterns and clean architecture principles.

## Tech Stack

**Backend:** Node.js + Express | RESTful API | Layered architecture (routes/controllers/services)  
**Frontend:** React 19 + Vite | Component-based architecture | State management with hooks

## Architecture Highlights

- **Separation of Concerns**: Routes → Controllers → Services → Data layer
- **Scalable Structure**: Ready for TypeScript migration and database integration
- **Professional Git Workflow**: Conventional commits, atomic changes
- **API-First Design**: Backend as standalone REST API
- **Code Quality**: ESLint with Airbnb standards, automated linting and fixing

## Setup

```bash
# Backend
cd backend && npm install && npm run dev

# Frontend  
cd frontend && npm install && npm run dev
```
